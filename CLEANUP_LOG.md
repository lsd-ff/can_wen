# 模拟与无用代码清理日志

- 完成日期：2026-07-23
- 审计范围：用户端后端 `backend`、管理端后端 `backend_mger`、用户端前端 `frontend`、管理端前端 `frontend_mger`、容器配置、脚本与项目文档
- 清理目标：删除旧的纯大模型模拟入口、硬编码演示数据、不可达实现、未使用声明与重复接口，同时保留并验证真实的 KG、HNSW、BM25 检索链路

## 一、清理原则

1. 领域问答不得绕过知识库直接调用大模型；统一进入四智能体编排链。
2. 删除生产代码中的硬编码模拟数据、历史占位实现、不可达分支和没有调用方的重复代码。
3. 不把测试替身、数据库映射、迁移契约、开发环境保护逻辑或第三方库 API 名称误判为模拟代码。
4. 每轮删除后执行后端测试、前端构建、静态检查、迁移检查和知识存储连通性验证。

## 二、已删除或收敛的代码

### 1. 用户端后端

- 删除旧的 `POST /api/v1/diagnosis/chat` 纯模型入口及其 410 占位路由；该路径现已完全退出路由和 OpenAPI，不再存在绕过知识库的入口。
- 删除 `DiagnosisChatRequest`、`DiagnosisChatResponse` 等旧接口模型，并把仍用于上下文表达的内部消息类型改为 `DiagnosisContextMessage`。
- 删除未使用的 `EventCallback`、重复的用户端管理内容/举报服务、社区举报审查服务及其请求和响应模型。
- 删除未使用的导入、临时变量、脚本字段和重复辅助函数。
- 测试改为直接断言旧路径不出现在 OpenAPI 中，防止该旁路以后回归。

### 2. 管理端后端

- 删除未使用的数据库健康辅助函数、列表/邀请/知识源模型、MinerU 冗余方法和模型网关内无调用方的 rerank 包装。
- 删除无调用方的术语辅助函数、知识重叠函数、旧风险事件 SQL 常量、字典转换函数及重复状态字段。
- 清理发布、任务、工作流、路由与服务模块中的未使用导入和无效 f-string。
- 保留模型注册表中的 rerank 配置，因为真实 Agentic Query 检索链仍会使用该能力。

### 3. 用户端前端

- 删除整套硬编码的长期记忆图谱演示页，包括固定病害节点、关系、统计数据、入口、路由和专属样式。
- 删除未实际使用的线程键和 API 类型。
- 将旧的 `diagnosis-simulated-banner` 样式名改为 `diagnosis-evidence-banner`；当前展示的是实际证据链状态，不是模拟结果。

### 4. 管理端前端

- 删除 `QueuePage` 在提前返回之后永远无法执行的旧页面实现。
- 删除无入口、无调用方的 `LegacyCommunityPage`。
- 删除未使用的 `SummaryChips` 组件及对应样式。

### 5. 配置与文档

- 更新用户端后端 README、前端 README、智能体架构文档和项目说明，统一描述真实四智能体链路及旧纯模型入口已删除的事实。
- 清理“原型、占位、未来补齐”等已失效描述，避免文档继续引导开发者使用旧实现。

## 三、审计中修复的运行问题

知识存储冒烟测试曾出现 Qdrant `502`。定位结果是 Windows 系统代理把本机 `127.0.0.1:6333` 请求转发到了代理端口，并非 Qdrant 服务故障。用户端和管理端的 Qdrant 客户端均已显式设置 `trust_env=False`，本机知识库请求不再读取系统 HTTP 代理。修复后 Redis、Qdrant、OpenSearch/BM25、Neo4j 四项连通性全部通过。

## 四、经审计后有意保留的内容

- 测试目录中的 Mock/Fake：仅用于隔离外部依赖，不进入生产运行路径。
- `d3ReheatSimulation()`：第三方图可视化库的力导向布局 API，不是业务模拟数据。
- `DISABLE_INSTALL_DEMO_CONFIG=true`：OpenSearch 官方容器安全设置，含义是禁用演示配置。
- 检索失败时的确定性降级：只基于已取得的证据生成保守答复，不伪造知识。
- 开发模式验证码：受开发环境开关保护，生产环境不启用。
- 初始 RBAC、模型注册和必要的 bootstrap 数据：属于系统初始化，不是问答模拟结果。
- ORM 模型、Alembic 元数据和原生 SQL 所依赖的表结构：属于持久化契约，即使静态工具无法看到动态引用也不能删除。
- 用户端已暴露且有测试覆盖的管理接口：属于真实业务 API，不作为重复模拟接口删除。
- 长期记忆：当前按产品计划保持关闭；本轮仅删除伪造的记忆图谱页面，不引入新的长期记忆实现。
